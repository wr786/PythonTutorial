# 简单运算

计算机，计算机，我们当然要学习怎么进行计算！

## 四则运算

```python
a = 7
b = 3
print(a + b)
print(a - b)
print(a * b)
print(a / b)
```

我们可以用`+-*/`来进行加减乘除的四则运算。其中需要注意的是，如果想要进行**整除**，我们需要使用`//`：

```python
print(a // b)
```

## 更多的运算

既然都知道整除了，不如再试试求余数：

```python
print(a % b)
```

幂运算：比如要求$a^b$，我们可以用`**`：

```python
print(a ** b)
```

求绝对值：这个需要用到函数`abs()`，如：

```python
print(abs(-1))
```

得到的结果是1。

## 字符串运算符

需要注意的是，`+`和`*`不仅能用于数值运算，还能用于字符串运算！比如

```python
a = "Yes?"
b = "Ha?"
print(a + b)
print(b * 3)
```

我们可以看到，`print(a + b)`输出的是`Yes?Ha?`而不是`Yes? Ha?`，`print(b * 3)`输出的是`Ha?Ha?Ha?`，只是将`b`重复了3次。
